## Features

This plugin can load and display 12 bit JPEG-images. It also supports windowing.

<img src="Screenshot1.png" alt="drawing" width="300"/>
<img src="Screenshot2.png" alt="drawing" width="300"/>

## Usage

```dart
Jpeg12BitWidget(
  input: img,
  windowMin: windowMin,
  windowMax: windowMax,
)
```
